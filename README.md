
# TP3 - Signal Processing 🎧

## 📌 Description

This project analyzes an audio signal using Fourier Transform.

## ⚙️ Features

* Generate audio signal (440Hz + 523Hz)
* Plot waveform
* Compute Fourier Transform
* Apply low-pass filter
* Extract features (RMS, Centroid, Bandwidth)

## ▶️ How to run

```bash
python tp3_signal.py
```

## 📊 Output

* waveform.png
* spectrum.png
* filter.png

## 🧠 Concepts

* Fourier Transform
* Signal filtering
* Spectral analysis
